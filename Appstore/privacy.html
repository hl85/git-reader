Gits Reader 隐私政策

更新日期：2026年6月19日

Gits Reader（以下简称“本应用”）是一款专为 Obsidian 和 Git 用户打造的 iOS 移动端笔记阅读器。我们非常重视您的隐私。本隐私政策旨在向您说明，本应用在运行过程中不收集、不存储、不传输您的任何个人信息或隐私数据。

1. 数据收集与使用

  • 个人信息：本应用不需要您注册账户，也不收集您的姓名、电子邮件地址、电话号码、设备标识符等任何个人信息。
  • 笔记数据：本应用是一款“本地优先（Local-First）”的应用。您通过 Git 同步的笔记数据全部存储在您 iOS 设备的本地沙盒（Sandbox）中。我们没有服务器，绝不会上传、读取或分析您的笔记内容。
  • 凭据安全：您在配置仓库时输入的 Git 个人访问令牌（PAT）仅用于与您指定的 Git 托管服务商（如 GitHub、Gitee 等）进行加密通信。该凭据通过 iOS 系统底层的 Keychain 进行安全存储，本应用绝不会将其发送给任何第三方。

2. 第三方服务

本应用不集成任何第三方广告、追踪器或分析 SDK（如 Google Analytics、Umeng 等）。本应用仅直接与您配置的 Git 远程仓库服务器进行通信，通信过程完全取决于您所使用的 Git 服务商的隐私政策。

3. 数据安全

由于所有数据均保存在您的本地设备上，您的数据安全主要取决于您设备的物理安全以及 iOS 系统的安全机制。我们建议您为设备设置锁屏密码或启用 Face ID / Touch ID 以保护您的本地数据。

4. 政策更新

我们可能会不时更新本隐私政策。任何修改都将在此页面上公布。建议您定期查看本页面以获取最新信息。

5. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过 GitHub Issues 与我们联系。
